# # NESKI Studio

Company website built with React, TypeScript and Vite.

## Overview

The official website for NESKI Studio, built with modern frontend technologies. The project also serves as a portfolio showcasing responsive design, clean UI and modern development practices.

## Tech Stack

- React
- TypeScript
- Vite
- TanStack Router
- Tailwind CSS
- Lucide React

## Getting Started

Install dependencies:

```bash

npm install

```

Start the development server:

```bash

npm run dev

```

Build for production:

```bash

npm run build

```

Preview the production build:

```bash

npm run preview

```
